Quick solution to game on openSUSE (Leap 15.6)
Use regata OS, it is openSUSE Leap base with latest kernel, and out of the box added all the needed repositories for multimedia, drivers, applications, Also Steam is installed by default. Customised KDE Plasma. it has a repo hosted on opensuse.org repositories, they use kernel 6.15 of the time of this video, the kernel is in regata repo. Regata has two ISOs, one without nvidia drivers pre-installaed, and the other contains Nvidia drivers

-----------------------------------------------------------------------------

now, Using Tumbleweed with gaming supported hardware:

Preparations:
------------
for DE (Gnome or KDE), Display Manager (Xorg or Wayland), Security (Apparmor or SELiux), it is your choice, whatever suits your needs

my setup, is KDE, X11, and Apparmor

1- Nividia GPU - RTX 2060 6GB + AMD Ryzen 7 4800H 8-Core
/etc/default/grub
Append 'rd.driver.blacklist=nouveau' to end of ‘GRUB_CMDLINE_LINUX=""

/etc/modprobe.d/50-blacklist-nouveau.conf
blacklist nouveau
options nouveau modeset=0

# dracut -f
# grub2-mkconfig -o /boot/grub2/grub.cfg

#### Bash Shell Environment Variables ###
__NV_PRIME_RENDER_OFFLOAD=1 __VK_LAYER_NV_optimus=NVIDIA_only __GLX_VENDOR_LIBRARY_NAME=nvidia QT_QPA_PLATFORM=xcb CUDA_VISIBLE_DEVICES=0 __GL_THREADED_OPTIMIZATIONS=1 __GL_SYNC_TO_VBLANK=0 __GL_ALLOW_UNOFFICIAL_PROTOCOL=1 __NV_PRIME_RENDER_OFFLOAD=1 __NV_PRIME_RENDER_OFFLOAD_PROVIDER=NVIDIA-G0 __VK_LAYER_NV_optimus=NVIDIA_only __GLX_VENDOR_LIBRARY_NAME=nvidia KDE_FULL_SESSION=1 XDG_CURRENT_DESKTOP=KDE XDG_SESSION_TYPE=x11

For Steam/games on X11 (PRIME render offload to NVIDIA):
        __NV_PRIME_RENDER_OFFLOAD=1
        __GLX_VENDOR_LIBRARY_NAME=nvidia
        __VK_LAYER_NV_optimus=NVIDIA_only

Optional (only if auto-detect fails):
        __NV_PRIME_RENDER_OFFLOAD_PROVIDER=NVIDIA-G0

For DaVinci Resolve (only if you’re on Wayland):
       QT_QPA_PLATFORM=xcb

for the first time, better start steam from terminal to see what is going on while steam finish downloading its client.

Later, In case you are on KDE, and started Steam by clicking on the icon in the application menue, and Steam window did not show up, edit the following file to be like this:

.local/share/applications/steam.desktop
[Desktop Entry]
Categories=Game;
Comment=Application for managing and playing games on Steam
Exec=env RADV_PERFTEST=aco STEAM_FORCE_DESKTOP_UI=1 mesa_glthread=true PROTON_ENABLE_FS_SYNC=1 /usr/bin/steam %U
Icon=steam
MimeType=x-scheme-handler/steam;x-scheme-handler/steamlink;
Name=Steam
NoDisplay=false
Path=
PrefersNonDefaultGPU=false
StartupNotify=true
StartupWMClass=Steam
Terminal=false
TerminalOptions=
Type=Application
X-KDE-SubstituteUID=false
X-KDE-Username=
